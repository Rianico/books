## Images 相关

1. 查看 Images：`docker images`
2. 只显示 Image ID：`docker images -q`
3. 删除多个 Image：`docker rmi $(docker images -q)`

## Container 相关

1. 查看运行中的 Container：`docker ps`
2. 查看所有 Container：`docker ps -a`
3. 查看所有 Container ID：`docker ps -a -q`
4. 停止所有 Container：`docker stop $(docker ps -a -q)`
5. 删除所有 Container：`docker rm $(docker ps -a -q)`
