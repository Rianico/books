## 常用工具

几个好用的工具网站：

- [Packages Search for Linux and Unix](https://pkgs.org/)：用于查找某个 rpm 包所在仓库
- [die.net](https://www.die.net/)：可以查看各个 Linux 组件的 man 信息
- [华为开源镜像站](https://mirrors.huaweicloud.com/)：提供了各个镜像仓库的国内加速地址，并且包含 arm 生态
