## 安装 Nodejs

```bash
# 设置华为国内镜像
npm config set registry https://mirrors.huaweicloud.com/repository/npm/
# 安装 Nodejs 版本控制工具
npm install -g n

# 删除原来的 npm 及 Nodejs
yum -y remove npm
yum -y remove nodejs

# 安装指定版本的 Nodejs
n install 12.0.0
# 查看版本
n ls
# 切换版本
n 10.23.0
```
