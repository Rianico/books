# IDEA 插件收藏

* **Save Actions:** 执行 save 操作的时候自动执行一些其他操作（如优化导入类）**。**
* **Maven Helper:** 可以用于协助解决依赖冲突的工具。
* **Git Commit Template:** 使用模板创建commit信息。
* **Rainbow Bracket**: 能够将括号以彩色显示，协助查看代码。
* **Translation**：翻译软件。
* **VisualVm Launcher：**VisualVm Launcher 插件，方便 debug JVM。
