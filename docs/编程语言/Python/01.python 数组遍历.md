# python 数组遍历

python遍历数组且带下标：

```python
_list = ['a', 'b', 'c', 'd', 'e']
# 使用enumerate
for idx, i in enumerate(_list):
    print('index：%s，value：%s' % (idx, i))
print('====================================')
# 使用range(start, end, step)
for idx in range(0, len(_list), 1):
     print('index：%s，value：%s' % (idx, _list[idx]))
```
