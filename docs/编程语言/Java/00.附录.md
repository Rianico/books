## Java 8 Stream

数组转 ArrayList、List

```java
int[] nums = new int[]{1, 2, 3};
// int[] 转 Object[] 数组
IntStream.of(nums).boxed().toArray();
// int[] 转 Integer[] 数组
IntStream.of(nums).boxed().toArray(Integer[]::new);

// int[] 转 List
IntStream.of(nums).boxed().collect(Collectors.toList());
// int[] 转 ArrayList
IntStream.of(nums).boxed().collect(Collectors.toCollection(ArrayList::new))
```

> NOTE： collect(Collectors.toList()) 方法只是表示返回一个 List 类型，但不保证具体实现类型，如果需要返回指定的 List 实现，使用 Collectors.toCollection(…) 方法。

## Gradle 国内镜像源替换

```groovy
repositories {
    mavenCentral()
  }
allprojects{
    repositories {
        def ALIYUN_REPOSITORY_URL = 'http://maven.aliyun.com/nexus/content/groups/public'
        def ALIYUN_JCENTER_URL = 'http://maven.aliyun.com/nexus/content/repositories/jcenter'
        def ALIYUN_APACHE_URL = 'https://maven.aliyun.com/repository/apache-snapshots'
        def ALIYUN_GOOGLE_URL = 'https://maven.aliyun.com/repository/google'
        def ALIYUN_SPRING = 'https://maven.aliyun.com/repository/spring'
        def ALIYUN_SPRING_PLUGIN = 'https://maven.aliyun.com/repository/spring-plugin'
        def ALIYUN_GRADLE = 'https://maven.aliyun.com/repository/gradle-plugin'
        all { ArtifactRepository repo ->
            if(repo instanceof MavenArtifactRepository){
                def url = repo.url.toString()
                if (url.startsWith('https://repo1.maven.org/maven2')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $ALIYUN_REPOSITORY_URL."
                    remove repo
                }
                if (url.startsWith('https://jcenter.bintray.com/')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $ALIYUN_JCENTER_URL."
                    remove repo
                }
                if (url.startsWith('https://repository.apache.org/')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $ALIYUN_APACHE_URL."
                    remove repo
                }
                if (url.startsWith('https://dl.google.com')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $ALIYUN_GOOGLE_URL."
                    remove repo
                }

            }
        }
        maven { url ALIYUN_REPOSITORY_URL }
        maven { url ALIYUN_JCENTER_URL }
        maven { url ALIYUN_APACHE_URL }
        maven { url ALIYUN_GOOGLE_URL }
        maven { url ALIYUN_SPRING }
        maven { url ALIYUN_SPRING_PLUGIN }
        maven { url ALIYUN_GRADLE }
        mavenCentral()
    }
}
```

## Maven 识别 Scala 项目

Maven 项目需要添加该插件，否则 Maven 打包时会识别不到 `src/main/scala` 路径下的代码：

```xml
<plugin>
    <groupId>net.alchim31.maven</groupId>
    <artifactId>scala-maven-plugin</artifactId>
    <version>4.5.3</version>
    <executions>
        <execution>
            <goals>
                <goal>compile</goal>
                <goal>testCompile</goal>
            </goals>
        </execution>
    </executions>
</plugin>
```
